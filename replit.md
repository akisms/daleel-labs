# Daleel Labs Site

## Overview
A static landing page website for Daleel Labs - a company offering AI automation and workflow orchestration services.

## Project Structure
- `index.html` - Main landing page with all content
- `server.py` - Simple Python HTTP server to serve the static files
- `Daleel Labs Site.rtf` - Original RTF file containing the HTML source

## Technology Stack
- HTML5
- Tailwind CSS (via CDN)
- Iconify icons (via CDN)
- Google Fonts (Inter, Crimson Pro)
- Python 3.11 (for static file serving)

## Running the Project
The project runs on port 5001 using Python's built-in HTTP server:
```bash
python server.py
```

## Features
- Responsive design with mobile-first approach
- Animated elements and blob backgrounds
- Interactive elements (toggles, hover effects)
- Clean, modern typography with serif/sans-serif combination

## Deployment
This is a static website that can be deployed as a static site.
